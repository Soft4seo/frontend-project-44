## To install the Brain Games
```sh
$ git clone https://github.com/Soft4seo/frontend-project-44.git
$ npm link
```

## To start "Brain even game"
```sh
$ brain-even

Ваша задача определить четное ли число. Ответьте «yes», если число четное, иначе ответьте «no».
```
